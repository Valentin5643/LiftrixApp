# Liftrix — Materiale pentru jurizare InfoEducație 2026

Acest folder conține materialele oficiale pentru evaluarea proiectului Liftrix.

## Fișiere principale

- `DOCUMENTATIE_TEHNICA_LIFTRIX.pdf` — documentația tehnică a proiectului
- `DECLARATIE_RESURSE_EXTERNE.txt` — resurse externe folosite și contribuția autorului
- `PREZENTARE_LIFTRIX_INFOEDUCATIE_2026.pptx` — prezentarea pentru susținere
- `APK/Liftrix-v1.0.0.apk` — APK pentru testare rapidă
- `dovezi/` — capturi pentru testare, performanță, securitate și stabilitate

## Rulare rapidă

1. Instalați APK-ul din folderul `APK/`.
2. Deschideți aplicația pe un dispozitiv Android compatibil.
3. Creați cont sau folosiți contul de test pus la dispoziție.
4. Testați fluxul principal: workout tracking → dashboard → AI coach/social.

# Liftrix - Materiale pentru jurizare InfoEducație 2026

Acest folder conține materialele pentru evaluarea proiectului Liftrix la secțiunea Software Utilitar / Mobile.

## Fișiere principale

- `DOCUMENTATIE_TEHNICA_LIFTRIX.docx` - documentația tehnică revizuită
- `DECLARATIE_RESURSE_EXTERNE.txt` - resurse externe folosite și contribuția autorului
- `PREZENTARE_LIFTRIX_INFOEDUCATIE_2026.pptx` - prezentarea pentru susținere
- `dovezi/` - folder pentru capturi de testare, performanță, securitate și stabilitate

## De completat înainte de predare

- Link public valid pentru repository.
- Link public valid pentru APK sau GitHub Release.
- Link public valid pentru demo video.
- Capturi reale din aplicație: sesiune activă, dashboard, social, AI coach, offline/sync.
- Capturi tehnice: test report, Firebase Rules, Crashlytics/Performance dacă se folosesc metrici.

## Rulare rapidă pentru juriu

1. Instalați APK-ul publicat pentru testare.
2. Deschideți aplicația pe un dispozitiv Android 8.0 / API 26 sau mai nou.
3. Creați cont sau folosiți contul de test pus la dispoziție.
4. Testați fluxul principal: workout tracking -> dashboard -> AI coach/social.

## Rulare rapidă pentru juriu

1. Descărcați APK-ul din linkul de release.
2. Instalați aplicația pe un dispozitiv Android 8.0 / API 26 sau mai nou.
3. Creați cont sau folosiți contul de test pus la dispoziție.
4. Testați fluxul principal: workout tracking -> dashboard -> AI coach -> social / QR buddy.

## Rulare din sursă

1. Clonați repository-ul.
2. Deschideți proiectul în Android Studio.
3. Adăugați `google-services.json` în folderul `app/`, dacă este necesar.
4. Rulați aplicația pe emulator sau dispozitiv fizic Android API 26+.

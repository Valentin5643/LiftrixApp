## Color Reference for AI

```
Primary Colors (Exact Hex Values):
- Night: #131515 (very dark gray-black)
- Jet: #2B2C28 (dark gray-green) 
- Persian Green: #339989 (medium teal-green)
- Tiffany Blue: #7DE2D1 (light mint-blue)
- Snow: #FFFAFB (very light pink-white)

Usage Guidelines:
- Light theme: 90% Snow backgrounds, accents in Persian Green/Tiffany Blue

- Minimize other colors - aim for 98% coverage with these 5 colors
```

## Tips for Best Results

1. **Specify both themes**: Ask for light and dark theme versions
2. **Mention Material 3**: Ensures modern Android design standards
3. **Include fitness context**: Helps AI understand the app type
4. **Request high contrast**: Emphasizes accessibility
5. **Add "minimalist"**: Reinforces the 5-color limitation
6. **Specify mobile/Android**: Ensures appropriate design patterns

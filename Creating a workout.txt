 Complete "Create Workout" UX Flow Design

  Entry Points & Initial Screen

  Main Entry Point: Floating Action Button (FAB)

  - Location: Bottom-right corner of Home Screen
  - Visual: Circular button with "+" icon, Material 3 primary color (#20C9B7)
  - Behavior: Taps to reveal workout creation options

  FAB Expansion Options

  When FAB is tapped, it expands to show two options:
  - "Create Workout" - Custom workout from scratch
  - "Use Template" - Start from existing template
  - Visual: Animated expansion with labels appearing beside smaller action buttons

  ---
  Flow 1: Create Custom Workout

  Step 1: Workout Setup Screen

  Screen Title: "New Workout"
  Back Button: Top-left, returns to Home Screen

  Main Content:
  - Workout Name Field
    - Placeholder: "e.g., Push Day, Morning Run"
    - Required field with subtle red indicator if empty
    - Max 50 characters with counter
  - Workout Type Selector
    - Default: "Strength Training"
    - Options: Strength Training, Cardio, Flexibility, Sports, Mixed
    - Visual: Horizontal scrollable chips
  - Target Duration (Optional)
    - Format: HH:MM with time picker
    - Placeholder: "Estimated duration"
    - Helper text: "This helps with scheduling"
  - Notes Field (Optional)
    - Placeholder: "Workout notes or goals..."
    - Expandable text area, max 200 characters

  Bottom Actions:
  - "Add Exercises" button (Primary, full-width)
  - "Save as Template" toggle (Off by default)

  Validation: Workout name is required before proceeding

  Step 2: Exercise Library Screen

  Screen Title: "Add Exercises"
  Subtitle: Shows current workout name
  Back Button: Returns to Workout Setup

  Search & Filter Bar:
  - Search Field: "Search exercises..." with search icon
  - Filter Chips: All, Upper Body, Lower Body, Core, Cardio, Equipment-based
  - Sort Dropdown: Alphabetical, Most Used, Recently Added

  Exercise List:
  - Each Exercise Card Shows:
    - Exercise name (e.g., "Barbell Bench Press")
    - Small muscle group tags (Chest, Shoulders, Triceps)
    - Equipment icon if applicable
    - "+" button to add to workout
  - Card Interaction:
    - Tap card body: Shows exercise details modal
    - Tap "+" button: Adds to workout with haptic feedback
    - Added exercises show checkmark and "Added" state

  Exercise Details Modal (when card is tapped):
  - Exercise name and description
  - Muscle groups targeted (visual diagram)
  - Equipment needed
  - Difficulty level (1-5 stars)
  - Proper form GIF or illustration
  - "Add to Workout" button at bottom

  Bottom Sheet: "Selected Exercises"
  - Shows count: "3 exercises selected"
  - Tap to expand and see list with remove options
  - "Continue" button (only enabled when ≥1 exercise selected)

  Step 3: Exercise Configuration Screen

  Screen Title: "Configure Exercises"
  Subtitle: Shows workout name
  Progress Indicator: "Step 2 of 3" or similar

  Exercise List (Reorderable):
  - Each exercise shows as an expandable card
  - Drag Handle: Left side for reordering
  - Exercise Header (when collapsed):
    - Exercise name
    - Current sets summary (e.g., "3 sets planned")
    - Expand/Collapse Arrow

  Expanded Exercise Configuration:
  - Sets Table:
    - Columns: Set #, Weight (lbs/kg), Reps, Rest
    - Default: 3 empty sets
    - Add Set button below table
    - Delete Set button (X) for each row when >1 set
  - Input Behaviors:
    - Weight: Numeric keypad, allows decimals (e.g., 135.5)
    - Reps: Numeric keypad, integers only
    - Rest: Time picker in MM:SS format, default 60 seconds
    - Auto-fill: Previous set values populate as suggestions
  - Quick Actions:
    - "Copy Previous" - Copies last workout's data for this exercise
    - "Use Template" - Shows common rep/set schemes (3x8, 5x5, etc.)
  - Exercise Options Menu (three dots):
    - Replace exercise
    - Add notes to exercise
    - Remove from workout

  Bottom Actions:
  - "Add More Exercises" (Secondary button)
  - "Preview Workout" (Primary button)

  Step 4: Workout Preview & Save

  Screen Title: "Workout Preview"
  Subtitle: Workout name with edit icon

  Workout Summary Card:
  - Total exercises count
  - Estimated duration
  - Target muscle groups (visual icons)
  - Workout type badge

  Exercise Summary List:
  - Each exercise shows: Name, sets x reps summary
  - Tap any exercise to quick-edit
  - Reorder still available with drag handles

  Bottom Section:
  - "Save as Template" toggle with explanation tooltip
  - Template Name Field (appears if toggle is on)

  Final Actions:
  - "Start Workout" (Primary) - Saves and launches active workout
  - "Save for Later" (Secondary) - Saves as draft
  - "Discard" (Text button) - Shows confirmation dialog

  ---
  Flow 2: Create from Template

  Step 1: Template Selection Screen

  Screen Title: "Choose Template"
  Back Button: Returns to Home Screen

  Template Categories:
  - My Templates (User-created)
  - Popular Templates (Community favorites)
  - Beginner Programs
  - Advanced Programs
  - Quick Workouts (15-30 min)

  Template Cards:
  - Template Name (e.g., "Push Pull Legs - Day 1")
  - Creator: "By You" or "By Liftrix" or community user
  - Stats: Duration, exercises count, difficulty
  - Muscle Groups: Visual icons
  - Rating: Stars if community template
  - "Use Template" button

  Search & Filter:
  - Search bar: "Find templates..."
  - Filter by: Duration, Muscle Groups, Equipment, Difficulty
  - Sort by: Popular, Recent, Rating, Duration

  Step 2: Template Customization Screen

  Screen Title: Template name with "Customize" subtitle
  Back Button: Returns to template selection

  Workout Header (Editable):
  - Workout Name: Pre-filled, user can edit
  - "Customize Name" button to modify
  - Original Template: Shows source with link

  Exercise List (Same as Custom Flow Step 3):
  - All exercises from template pre-configured
  - User can modify sets, reps, weights
  - Add/remove exercises
  - Reorder as needed

  Template Modifications Indicator:
  - Shows "Modified from original" if changes are made
  - "Reset to Template" option available

  Quick Start Option:
  - "Start Workout Now" - Uses template as-is
  - "Customize First" - Enters customization mode

  Step 3: Save Options

  Same as Custom Flow Step 4, but with additional options:
  - "Update My Template" (if user owns template)
  - "Save as New Template" (always available)
  - "Use Once" (doesn't save modifications)

  ---
  Key UX Principles & Behaviors

  Progressive Disclosure

  - Start simple (name + type), add complexity gradually
  - Collapsible sections to reduce cognitive load
  - Smart defaults to minimize required input

  Data Persistence

  - Auto-save draft every 30 seconds
  - Warn before losing unsaved changes
  - Remember user preferences (units, default rest times)

  Input Validation

  - Real-time validation with helpful error messages
  - Required fields clearly marked
  - Sensible input limits (weight: 0-9999 lbs, reps: 1-999)

  Accessibility

  - All interactive elements have 44dp minimum touch targets
  - Proper content descriptions for screen readers
  - High contrast for text and important UI elements
  - Support for large text sizes

  Performance Considerations

  - Exercise library loads incrementally (20 at a time)
  - Search debounced by 300ms
  - Images lazy-loaded in exercise details

  Error Handling

  - Graceful offline behavior (show cached exercises)
  - Clear error messages for network issues
  - Retry mechanisms for save operations

  This flow balances flexibility with simplicity, allowing both quick template usage and detailed custom workout creation while maintaining a consistent, intuitive user experience 
throughout.
